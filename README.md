# Backend сервер

Backend на **Node.js + Express**.

## Требования

- [Node.js](https://nodejs.org/) (рекомендуется LTS-версия)
- npm
- База данных **PostgreSQL** (например, через [Neon](https://neon.tech), Supabase, или локальный PostgreSQL)

## Установка

1. Клонируйте репозиторий:

   ```bash
   git clone <ссылка на репозиторий>
   cd <папка проекта>
   ```

2. Установите зависимости:

   ```bash
   npm i
   ```

## Настройка базы данных

1. Создайте свою базу данных PostgreSQL (например, на [neon.tech](https://neon.tech) — бесплатно и быстро).
2. В корне проекта создайте файл `.env` (или измените существующий) и укажите данные **своей** базы данных:

   ```dotenv
   DB_NAME=ваша_база
   DB_USER=ваш_пользователь
   DB_PASSWORD=ваш_пароль
   DB_HOST=ваш_хост
   DB_PORT=5432
   ```

   > ⚠️ Не используйте чужие или тестовые учётные данные. Файл `.env` не должен попадать в git (добавьте его в `.gitignore`).

## Запуск

### Режим разработки (с автоперезапуском)

```bash
npm run dev
```

### Обычный запуск

```bash
node index.js
```

После запуска сервер автоматически подключится к базе данных, используя параметры из `.env`. Дополнительных действий не требуется.
